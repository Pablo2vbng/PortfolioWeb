#  Portfolio Web - Pablo Vidal Vidal

**Full Stack Developer** | Soluciones modernas para Pymes y Autónomos.

Este repositorio contiene el código fuente de mi sitio web personal y portafolio profesional. El proyecto está diseñado para mostrar mi experiencia, las tecnologías que domino y una selección de mis trabajos más destacados en el desarrollo web.

##  Descripción del Proyecto

Este portafolio es una SPA (Single Page Application) estática, diseñada con un enfoque **Mobile First** y totalmente **Responsive**. Sirve como punto de contacto central para clientes potenciales y reclutadores.

### Características Principales:
- **Diseño Responsivo:** Adaptable a móviles, tablets y escritorio gracias a **Bootstrap 5**.
- **Navegación Fluida:** Menú de navegación fijo con scroll suave a las secciones.
- **Showcase de Proyectos:** Tarjetas interactivas con enlaces a demos y repositorios.
- **Sección de Tecnologías:** Visualización de mi stack técnico (AWS, Angular, React, Python, etc.).
- **Formulario de Contacto:** Interfaz limpia para la captación de leads.

##  Tecnologías Utilizadas

El sitio ha sido construido utilizando las siguientes tecnologías:

*   **HTML5:** Estructura semántica.
*   **CSS3:** Estilos personalizados (`css/styles.css`) y diseño visual.
*   **Bootstrap 5.3.8:** Framework para el sistema de rejilla (grid), componentes (Navbar, Cards, Modal) y responsividad.
*   **Bootstrap Icons:** Iconografía vectorial.
*   **JavaScript:** Lógica de interacción (si aplica en futuras expansiones).

Al ser un sitio estático, no requiere instalación de dependencias de backend complejas para visualizarse.

Para ver en local:
Clona este repositorio:
